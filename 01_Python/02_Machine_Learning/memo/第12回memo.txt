探索的データ解析
どのようなデータで学習をさせるか吟味する


半教師あり学習
一部のデータにのみ正解（ラベル）が与えられている問題。
少量のラベル付きデータを使って、その他大量のラベルなしデータも識別できないか？

→決定境界で分類がスパッと分かれているデータは半教師あり学習に向いている。

半教師あり学習の一つ「ラベル伝播法」
データ間の類似度にもとづきグラフを作っておく。（数学のグラフ理論。ノードとエッジ）
「隣り合ったノードからノードへラベルが伝播していく」
（決定境界が入り組んでいるのは向いていない。単純なやつなら。）
LabelPropagation



＜探索的データ解析＞データの特徴を見つけ、半教師あり学習に結びつけるため

from sklearn.semi_supervised import LabelPropagation
# LabelPropagationクラスでオブジェクトを作り、.fitで学習させる。

ラベル伝播=LabelPropagation(max_iter=10000) 	# 最大繰り返し数
ラベル伝播.fit(説明変数X,ラベル配列) 		# 学習
予測されたy=ラベル伝播.predict(説明変数X)		# predictで学習結果を取り出す

LabelPropagationクラスの.scoreというメゾッドをよぶと、正解率を出してくれる
